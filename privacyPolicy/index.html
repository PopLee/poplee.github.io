隐私政策

生效日期：2026年2月

云弧投研（以下简称「本应用」）由云弧（CloudArc）提供。我们重视您的隐私，本政策说明我们如何对待您的数据。

一、数据存储原则（本地优先）

1.1 本应用采用「本地优先」设计。您的以下数据默认仅存储在您使用本应用的设备本地（如策略配置、回测记录、数据管理设置、API 配置等），我们不会主动收集或上传至我们的服务器。

1.2 本地数据存储采用本机数据库（如 SQLite）及本地文件，您可自行备份或删除。

二、我们可能接触到的数据

2.1 您主动填写的 API 密钥（如数据源 API、大模型 API 等）：仅保存在您设备本地，用于本应用向相应服务发起请求。我们不会收集、存储或查看您的 API 密钥。

2.2 当您使用需要网络的功能时（如行情数据、新闻、AI 分析等），本应用会向以下类型的第三方发送请求，具体以您配置的数据源为准：

  • 数据源：公开接口或您配置的数据接口；
  • 大模型服务：如您自行配置的 LiteLLM / 大模型兼容接口。

上述请求由您的设备直接发往相应服务商，其数据处理遵循各服务商之隐私政策。

2.3 我们不会收集您的姓名、邮箱、手机号、设备标识等个人身份信息，除非您在未来使用需账号登录的功能并自愿提供。

二之一、第三方大模型（AI）服务（数据披露与同意）

当您使用本应用中的 AI 对话、AI 分析等功能时：

  • 我们会将您输入的对话内容、消息文本及相关上下文发送至您在本应用「模型设置」中所配置的大模型服务商（如 DeepSeek、通义千问等，具体以您配置为准）。上述数据由您的设备直接发往该服务商，我们不经手存储或转发。
  • 我们会在您首次使用 AI 对话前明确说明上述数据发送对象与用途，并在您同意后再发送。
  • 我们不会控制上述第三方服务商对数据的使用与保留，其数据处理以该服务商的隐私政策为准。我们建议您在使用前查阅所配置服务商的隐私政策。
  • 本政策其他条款中关于「我们可能接触到的数据」「数据来源与合规」等同样适用于上述场景。
简要说明：使用 AI 功能时，您的输入会发往您自行配置的大模型服务商；应用内会在首次使用前征得您的同意，并可在设置中查看完整隐私政策。

三、数据来源与合规

3.1 行情、资讯等数据可能来自第三方数据源（包括但不限于公开接口或您配置的数据服务）。使用这些数据时，请您遵守相关数据源的使用条款与合规要求。

3.2 本应用不向第三方出售您的个人数据，也不将本地存储的数据用于广告定向或与无关第三方共享。

四、权限与安全

4.1 本应用仅在您授权或系统要求范围内使用网络、本地文件访问等权限，用于实现数据下载、导出、配置存储等功能。

4.2 我们建议您妥善保管本机存储的 API 密钥及本地数据，避免设备丢失或未授权访问带来的风险。

五、政策更新与联系

5.1 我们可能适时更新本政策，更新后的版本将在应用内或官网发布。继续使用本应用即视为接受更新后的政策。

5.2 如您对本政策有疑问，可通过邮箱（cloudarcquant@gmail.com）与我们联系。
