ArcticFoxAlpha 隐私政策

生效日期：2026 年 3 月

ArcticFoxAlpha（以下简称“本应用”）重视用户隐私。本政策用于说明我们如何处理与保护您的数据。

本应用定位为 iPad / iOS 方向产品，采用本地优先架构；用户在应用中配置的数据源与 AI 服务密钥由用户自行管理， 联网请求由设备直接发往用户选择的第三方服务商。

一、数据存储原则（本地优先）

  1. 本应用采用“本地优先”架构。您的策略配置、回测记录、模型配置、数据源设置等信息默认保存在设备本地（如 SQLite / 本地文件）。
  2. 我们不会主动将上述本地数据上传至我们的服务器，除非未来新增需要云同步的功能并取得您的明确同意。
  3. 您可在本地自行备份、导出或删除相关数据。

二、我们可能接触的数据

  1. 您主动填写的数据：例如数据源 API Key、大模型 API Key，仅用于由您的设备直接向第三方服务发起请求。
  2. 网络请求数据：当您使用行情、新闻、AI 分析等联网功能时，应用会将必要请求参数发送至您配置的第三方服务商。
  3. 诊断信息（可选）：若您主动开启调试或提交问题反馈，可能包含错误日志、请求状态码、时间戳等技术信息，不包含您主动输入的敏感正文内容（除非您在反馈中自行提供）。

三、第三方 AI 服务说明（重要）

  1. 当您使用 AI 对话、AI 解读等功能时，您的输入文本及必要上下文会发送至您在应用内配置的大模型服务商。
  2. 相关请求由您的设备直接发送，我们不托管您的 API Key，不代存您的对话正文。
  3. 第三方服务商的数据处理规则适用其自身隐私政策与服务条款，请您在使用前审阅。

四、数据来源与合规

  1. 行情、资讯等数据可能来自第三方数据服务（例如您配置的数据接口）。
  2. 您应遵守第三方数据服务的使用条款、授权范围与合规要求。
  3. 本应用不向第三方出售您的个人数据，不用于广告定向，不与无关第三方共享本地业务数据。

美股信息服务边界说明

  1. 本应用提供的信息与 AI 输出仅用于信息参考与研究辅助，不构成投资建议、收益承诺或保证。
  2. 金融市场存在高风险，您应独立判断并自行承担决策后果。
  3. 因第三方数据源中断、延迟、错误、限流或权限变化导致的功能异常，本应用不对由此产生的交易损失承担责任。

五、权限与安全

  1. 本应用仅在实现功能所必需范围内使用网络、本地文件访问等权限。
  2. 请妥善保管设备与本地密钥，避免未授权访问。
  3. 若设备丢失、系统被入侵或您怀疑密钥泄露，请立即更换相关第三方服务密钥。

六、未成年人保护

若您未满 18 周岁，请在监护人指导下阅读并使用本应用。

七、政策更新

  1. 我们可能根据产品迭代或法律要求更新本政策。
  2. 更新版本将通过应用内或项目页面发布。
  3. 您继续使用本应用即视为同意更新后的政策。

八、联系我们

如对本政策有疑问，请通过邮箱（cloudarcquant@gmail.com）与我们联系。
