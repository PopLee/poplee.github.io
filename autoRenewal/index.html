一、会员自动续费服务说明

1.1 会员自动续费服务分为连续包月、连续包季、连续包年等类型（具体以应用内订阅页面展示为准）。如您开通本服务，即视为您授权我们有权在您的云弧量研 Premium 会员服务即将到期时，从您用于支付的 Apple ID（iTunes/App Store 账户） 中自动扣取下一计费周期的费用。

1.2 本服务实现的前提是您已使用有效的 Apple ID 完成首次购买，且该账户在续费时可成功扣款。您有义务确保该账户在续费时具备足够余额或已绑定有效支付方式。如因余额不足、支付方式失效等原因导致扣款失败，我们有权在不再另行通知的情况下暂停或终止您的会员权益，直至您成功续费或重新订阅。

1.3 若您希望取消自动续费，须在当前订阅周期结束前至少 24 小时关闭自动续费，否则系统仍将按周期自动续订并扣款。非因您违反法律法规或本规则，您可随时关闭自动续费；已开通的当前周期（当月/当季/当年）费用不予退还。

二、如何取消自动续费

若您选择自动续费服务，可通过以下方式中止或终止本服务。未在订阅期到期前至少 24 小时取消自动续费的，将会自动续订并扣款。

2.1 macOS（Mac 电脑）— App Store 订阅

  • 打开 App Store → 点击左下角 「商店」 → 「查看我的账户」（或点击账户头像）
  • 进入 「账户设置」 / 「我的账户信息」 页面 → 找到 「订阅」 或 「管理」
  • 选择 「云弧量研」 或 CloudArc Quant → 点击 「取消订阅」 或 「关闭自动续期」

2.2 iOS（iPhone / iPad）— App Store 订阅

  • iOS 12 及以上：打开 App Store → 点击右上角 Apple ID 头像 → 「账户」 → 「订阅」 → 选择 云弧量研 → 取消订阅
  • iOS 12 以下：打开 「设置」 → 「iTunes Store 与 App Store」 → 点击 Apple ID → 「查看 Apple ID」 → 「订阅」 → 选择 云弧量研 → 取消订阅

取消后，您在当前已付费周期内仍可继续使用会员权益，到期后将不再自动扣款。

我们有权根据产品与运营需要，对会员自动续费服务的价格、周期或权益内容进行调整，并将在应用内或官网进行说明。若您不同意变更，可在当前周期结束前关闭自动续费。
